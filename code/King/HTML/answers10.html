2.

(a) b, c, and d are visible. Local b hides external b.
(b) a, b, and c are visible. Local c hides external c.
(c) a, b, c, and d are visible. Block a hides parameter a.
(d) b, c, and d are visible. Local c hides external c.

3. The program could contain an external variable named i, a local variable or parameter named i, and an unlimited number of blocks containing a declaration of i: 

int i;   /* external variable */

int main(int i1, char* argv[])   /* argc parameter */
{
  int i2;    /* local variable (only one of i1 and i2 can be 'i') */

  {
    int i;   /* first block variable */
  }

  {
    int i;   /* second block variable */
  }

  {
    int i;   /* and so forth... */
  }

  return 0;
}
